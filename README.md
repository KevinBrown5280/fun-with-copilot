# fun-with-copilot

A personal sandbox for exploring and experimenting with GitHub Copilot.

## Playbooks

| Playbook | Description |
|----------|-------------|
| [multi-model-debate](playbooks/multi-model-debate/) | **GitHub Copilot CLI** — Pit multiple AI models against each other to stress-test a technical decision |
| [adversarial-review](playbooks/adversarial-review/) | **GitHub Copilot CLI** — 4-model independent code review with voting reconciliation and cross-session dismissed-findings persistence |

See each playbook's README for setup and usage instructions.
